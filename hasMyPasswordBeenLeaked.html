Has my password been leaked?

Want to see how your password fairs against a database of over 600 million passwords known to be exposed in data breaches? Enter a password below to securly compare them against the database.

Upon submission, the password provided will be hashed and used in an API call to Pwned Passwords. If the password provided is found to have been leaked in a known data breach, the Password Breach Investigator will indicate to the user how many breaches the password was found in.

If the password entered appears to have been found in a multitude of breaches, the Make Hackers Mad team strongly suggests changing your password to a more secure one through the help of our password generation tool.
